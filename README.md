# 📌 Internship Management Portal – Internnova

A full-stack web application designed to streamline the internship process for **students** and **faculty/admins**. The platform allows students to apply for internships, track applications, and manage profiles, while faculty can manage listings, approvals, and monitor student progress.

---

## 🚀 Features

### 👨‍🎓 Student Dashboard
- View available internships
- Apply for internships
- Track application status
- Manage profile
- Upload documents (Resume, NOC, etc.)
- Notification system

### 👨‍🏫 Faculty/Admin Dashboard
- Create and manage internship listings
- Approve/Reject applications
- NOC approval system
- Monitor student activities
- Dashboard analytics (optional)

### 🔐 Authentication
- Secure Login/Register
- Role-based access (Student / Faculty)
- Protected routes

### 📊 Additional Features
- Responsive UI
- Sidebar navigation
- Real-time updates (optional)
